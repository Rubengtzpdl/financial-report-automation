# Financial Report Automation

Automated pipeline that generates professional PDF financial reports from raw Excel data using Python.

## What it does
- Reads raw financial statements (Excel)
- Calculates key financial indicators (margins, EBITDA, growth)
- Generates charts automatically
- Produces a formatted PDF report — all with one command

## Tech Stack
- **Python** — automation pipeline
- **Pandas** — data processing
- **Matplotlib & Seaborn** — visualizations
- **ReportLab** — PDF generation
- **Git/GitHub** — version control

## Run everything with one command
```bash
python scripts/run_all.py
```
